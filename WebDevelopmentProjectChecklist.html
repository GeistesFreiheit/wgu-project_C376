Project in Web Development Fundamentals (UG, YZT-0616)

  1. Site Template
    1. Include the following Semantic Elements in each of your HTML documents:
      1. Header learn about the header tag
        1. Do you have the semantic <header> element within the <body> of your page?
        2. Does your <header> contain a statement describing the overall site topic?
      2. Nav learn about the nav tag
        1. Do you have the semantic <nav> element wrapping your navigation links?
        2. Do you have links on each page that take you to all other pages?
      3. Section learn about the section tag
        1. Is your main content wrapped in the semantic <section> element?
        2. Is the <section> located in the same place on each page?
      4. Aside learn about the aside tag
        1. Is your secondary content wrapped in the semantic <aside> element?
        2. Is the <aside> present on each page?
        3. Is the <aside> located in the same place on each page?
        4. Is the content within the <aside> relevant to that page's topic?
    2. Design your template with a consistent look and feel by using each of the following elements:
      1. External Style Sheet learn about the link tag
        1. Is all of your CSS code stored in an external style sheet?
        2. Does every page link to the same style sheet?
        3. Have you validated your CSS code to ensure there are no errors?
      2. Float Property learn about the float property
        1. Have you used the float property within your CSS stylesheet?
        2. Does the application of the float property result in two objects arranged side-by-side?
      3. Absolute Positioning learn about absolute positioning
        1. Have you used position:absolute in your CSS stylesheet?
        2. Have you used an offset property (top, right, bottom, left) to adjust where the element is being positioned?
      4. Relative Positioning learn about relative positioning
        1. Have you used position:relative in your CSS stylesheet?
        2. If you remove position:relative from your stylesheet do you see a visible change in how the page renders?
      5. :hover Pseudo Class learn about the hover selector
        1. When you move your mouse over the navigation links is there a visible change to their appearance?
      6. Articulation of Response (clarity, organization, mechanics)
        1. Did you spell check each page?
        2. Is your text content written in grammatically correct sentences?
  2. Create at least four web pages of content for your website, using the template from part A.
    1. Include the following HTML items in the content of your website:
      1. Title learn about the title tag
        1. Is the <head> of the document have you used the <title> element on each page? (note, this is different than the <header> element from section A.
        2. Is your <title> unique on each page?
        3. Does your <title> describe what each page's topic is?
      2. Images learn about the image tag
        1. Does your site include at least one <img>?
        2. Does each image have an ALT attribute that describes what the image represents?
      3. Links learn about the anchor tag
        1. Do you have a link that directs you to another website?
        2. Does that link utilize the TARGET attribute?
        3. When you click the link does it open in a new window or tab?
      4. One Ordered List learn about ordered lists
        1. Have you used on ordered list within your site?
        2. Does the ordered list have at least 3 list items?
      5. One Unordered List learn about unordered lists
        1. Have you used on unordered list within your site?
        2. Does the unordered list have at least 3 list items?
      6. One Table learn about tables
        1. Does your site have at least one <table>?
    2. Include the following CSS rules to enhance your website content:
      1. Color and Font Changes
        1. Have you changed the appearance of fonts at least once within your CSS stylesheet? (this could include color changes, font-family, font-weight, font-size, etc.)
      2. Type Selector learn about type or element selectors
        1. Does your CSS stylesheet contain at least one rule that target an HTML element using the tag's name (e.g., p, table, th, td, ul, ol, li, h1-h6, a)
      3. Class Selector learn about class selectors
        1. Does your CSS stylesheet contain a rule for a class selector?
        2. Have you applied the that class selector at least two times within the HTML of your site?
      4. ID Selector learn about ID selectors
        1. Does your CSS stylesheet contain at least one rule for an ID selector?
        2. Have you applied that ID selector at least once within the HTML?
